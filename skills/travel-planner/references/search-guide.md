# WebSearch 搜索策略

## 搜索时机

当 opencli 命令不可用或需要补充信息时使用 WebSearch。

## 平台指向词

不同平台使用不同的关键词组合可以提高命中率：

| 平台 | 指向词 | 示例搜索 |
|------|--------|----------|
| 大众点评 | 本地人推荐/苍蝇馆子/排队/小破店 | "绍兴 本地人推荐 苍蝇馆子" |
| 小红书 | 探店/出片/宝藏/避雷 | "绍兴 探店 宝藏小店" |
| 马蜂窝/穷游 | 攻略/路线/游记 | "绍兴 一日游攻略 路线" |
| 微信公众号 | 攻略/推荐/合集 | "绍兴 美食攻略 合集" |

## 补充策略

opencli 已覆盖主要平台（dianping/xiaohongshu/bilibili/weixin），WebSearch 用于：
1. 游记攻略（马蜂窝/穷游/Trip.com）
2. 补充验证（用不同搜索词交叉确认）
3. 发现 opencli 可能漏掉的新店铺
